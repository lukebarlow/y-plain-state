Yubiquity

This page demonstrates yubiquity, which synchronises arbitrary JavaScript data structures across browsers. To try it out, you'll need to open up the browser dev tools and type JavaScript commands directly. There is a global object set up called state. You can manipulate this by typing regular JavaScript things like

state.foo = 'bar'

or

state.children = []
state.children.push({ name : 'first child'})

The state will be updated in the box below, and in any other browser looking at the same page.
